== Description ==

My Twitter is a WordPress plugin made by Xhanch Studio to show your latest tweets, replies, user's profile picture, post date and click-able links within the tweets.

Once this plugin installed, My Twitter will be listed on your widgets so you can determine where to place it. The widget contains these settings

1. Title: the title of the widget
2. Name: your name
3. Username: username of your twitter account
4. # Latest Tweets: maximum number of tweets to show
5. Date Format: hide the date or display the date in a preferable format

By default, the layout will inherit your WP theme's style. But if you want to make it looks different, you can modify the CSS code for this plugin (located on this plugin's folder)

== Installation ==

1. Download and unzip the package.
2. FTP the entire "my-twitter" directory to your /wp-content/plugins/ directory on your blog.
3. Activate the plugin on the "Plugins" tab of the administartion panel.
4. Put "My Twitter" widget to your sidebar.
5. Open the widget options and put in your details

By default, the layout will inherit your WP theme's style. But if you want to make it looks different, you can modify the CSS code for this plugin (located on this plugin's folder)
